# Sweet Crave Corner
A simple Bakery Management System built in C for managing products, customers, and sales records.  
This project was developed as part of the Programming Fundamentals course.

## Features
- Add, delete, and update bakery products  
- Customer registration and login system  
- Place orders and generate receipts  
- File handling for storing products, customers, and sales data  

## Technologies Used
- C programming language  
- File handling (`stdio.h`) for data storage  
- Arrays for managing customer and product records  

## How to Run
1. Clone or download the repository.  
2. Open the project in any C IDE (e.g., Dev-C, Code::Blocks, Turbo C).  
3. Compile and run `Final PF Project.c` (or `bakery.c`).  
4. Follow the on-screen menu to interact with the system.  

## Project Structure
- `Final PF Project.c` / `bakery.c` → Main source code  
- `README.md` → Project description  
- `products.txt`, `customers.txt`, `sales_log.txt` → Data files created during runtime  

## Purpose
This project demonstrates the use of **file handling, arrays, and functions** in C by simulating a real-world bakery management system.
